# 🏕️ Outera - Aplikasi Penyewaan Alat Camping

Outera adalah aplikasi Android yang dirancang khusus untuk mempermudah proses penyewaan perlengkapan berkemah. Dengan antarmuka yang modern, elegan, dan *user-friendly*, aplikasi ini memungkinkan pengguna untuk melihat katalog alat, melakukan pemesanan, melacak riwayat sewa, hingga memantau laporan statistik penyewaan.

Aplikasi ini dikembangkan sebagai pemenuhan **Proyek MID Pemrograman Mobile**.

---

## ✨ Fitur Utama

1. **Dashboard (Beranda):** Menampilkan ringkasan status penyewaan aktif dan ketersediaan stok alat kemah.
2. **Katalog Alat:** Menampilkan daftar lengkap perlengkapan camping dengan visual gambar berkualitas dan fungsi pencarian.
3. **Formulir Sewa:** Formulir dinamis untuk menyewa alat, dilengkapi fitur pemilih tanggal (*Date Picker*) dan perhitungan harga otomatis berdasarkan durasi pinjam.
4. **Riwayat & Pengembalian:** Memantau semua transaksi aktif dalam bentuk *Bento Card* dengan fitur "Selesaikan Sewa" untuk mengembalikan barang dan menghapus daftar aktif.
5. **Laporan Statistik:** Menyajikan grafik batang interaktif yang bisa digeser (*Horizontal Scroll*) dan kalkulasi total pendapatan penyewaan secara *real-time* dari riwayat permanen.

---

## 🚀 Cara Menjalankan Proyek

1. Pastikan Anda telah menginstal **Android Studio** versi terbaru.
2. Lakukan *clone* repositori ini melalui terminal:
   ```bash
   git clone https://github.com/Hanifzx/outera-android-app.git
   ```
3. Buka folder repositori proyek ini di dalam Android Studio.
4. Tunggu beberapa saat hingga proses sinkronisasi Gradle (Build) selesai otomatis.
5. Tekan tombol **Run** (atau `Shift + F10`) untuk menjalankan aplikasi di Emulator bawaan Android Studio atau hubungkan ke *smartphone* fisik Anda melalui USB Debugging.
